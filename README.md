## how to run project


* clone from git
`git clone git@gitlab.com:serverscom/test-project.git`

* install npm libraries

`npm install`

* run server

`npm start`

* open browser 

`http://localhost:8000`

* see demo

`https://newsmodule.herokuapp.com`


## server requirements
